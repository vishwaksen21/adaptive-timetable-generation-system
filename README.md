# Adaptive Timetable Generation System

## Setup Guide

1. Open Terminal
2. Clone and enter the repository:
```bash
cd ~/Documents/GitHub
git clone https://github.com/yourusername/adaptive-timetable-generation-system.git
cd adaptive-timetable-generation-system
```

3. Run setup script:
```bash
chmod +x setup.sh
./setup.sh
```

4. Activate the environment:
```bash
source venv/bin/activate
```

5. Run the system:
```bash
python main.py
```

## Requirements
- Python 3.7+
- pandas
- numpy
